# tex.freeslot-schedule.com snapshot

この repo は tex.freeslot-schedule.com の再現に必要な source / config のスナップショットである。

## 含めるもの
- texcloud-plus 拡張ソース
- docker compose 類
- nginx 設定
- viewer.html や tasks.json など補助ファイル
- code-server 拡張一覧

## 含めないもの
- Let's Encrypt 証明書
- code-server の実パスワード
- 個人の workspace 本文ファイル
- .env / token / secret 類
- node_modules / build artifact
